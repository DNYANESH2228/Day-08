# Day-08
Data Breach Awareness &amp; Threat Intelligence Basics
1️⃣ Breach Search Observation

I used the website HaveIBeenPwned to check whether an email address was involved in any data breach.

The website shows if the email was found in a breached database. It also shows the name of the breached website, the year of the breach, and the type of information that was exposed.

The exposed data may include email addresses, passwords, usernames, or other account details. This helps users know if their personal data has been leaked and if they should change their passwords.
